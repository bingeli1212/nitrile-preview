---
title: The "verb" paragraph
latex.parskip: 1
---

The "verb" paragraph is recognized by the "verb" keyword
after the paragraph fence.

~~~~framed
~~~verb
function func1(a, b, c) {
  console.log(arguments[0]);
  // expected output: 1

  console.log(arguments[1]);
  // expected output: 2

  console.log(arguments[2]);
  // expected output: 3
}
~~~
~~~~
